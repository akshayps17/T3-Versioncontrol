## Workflow
1. Create feature branch from `dev`
2. Commit & push changes
3. Open Pull Request → Merge to `dev`
4. Test and merge `dev` → `main`

## Tools Used
- Git
- GitHub

## Commands Used
```bash
git init
git branch -M main
git checkout -b dev
git checkout -b feature/login-module
git add .
git commit -m "Commit message"
git push origin branch-name
